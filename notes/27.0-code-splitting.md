27.0 Бывают проекты, которые достаточно тяжеловесны даже после сжатия и минификации файлов. Чтобы ускорить загрузку приложения, мы можем применить динамический импорт функций, загружая их только тогда, когда они действительно понадобятся. Например, таким образом:

import("./math").then(math => {
    console.log(math.add(16, 26))
})
