// ? 12.0 «Redux Toolkit» — это глобальное хранилище данных для веб-приложения. Когда приложение становится довольно объёмным, то использовать только useState & useContext становится уже недостаточным и неудобным. Redux имеет ряд преимуществ перед обычным React Context, например он делает оптимизацию, чтобы при изменении каких-то компонентов, которые следят за общими глобальными данными ререндерился только активный компонент, а не все, как это происходит в случае с React Context. Хотя по принципу действия и целям они очень похожи, по сути Redux это более продвинутая надстройка на базе оригинального функционала React Context для обеспечения единого глобального хранилища данных приложения. React Context неплох для маленьких приложений и справляется со своими задачами, но его сложно масштабировать и для серьёзных проектов его заменяют Redux.
