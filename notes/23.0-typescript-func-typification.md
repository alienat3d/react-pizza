23.1 Дальше рассмотрим типизацию внутри непосредственно стрелочной функции на примере строки ввода "input". Здесь мы типизируем функцию, которая обрабатывает слушатель события "change" на DOM-элементе "input". Правильное название элемента для типизации можно подсмотреть во всплывающей подсказке, наведя на этот элемент в вёрстке. Также, для события клик это будет MouseEvent и т.д.

const onChangeInput = (evt: React.ChangeEvent<HTMLInputElement>) => {
    setValue(evt.target.value);
    updateSearchValue(evt.target.value);
}
