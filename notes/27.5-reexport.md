27.5.0 Что значит "Reexport"? Это техника сокращения импортов компонентов для удобства. Например, мы могли бы создать в папке [src/components] файл [index.ts] импортировать в него файлы из этой папки и снова экспортировать отдельные компоненты:

// [/src/components/index.ts]:

import CartItem from "./CartItem";
import Categories from "./Categories";
import Header from "./Header";
import Sorting from "./Sorting";
import PizzaBlock from "./PizzaBlock";
import Skeleton from "./PizzaBlock/Skeleton.tsx";
import Pagination from "./Pagination";

export {
    CartItem,
    Categories,
    Header,
    Sorting,
    PizzaBlock,
    Skeleton,
    Pagination
}

27.5.1 Затем, например, в комп. Home.tsx мы можем сократить импорт с:

// Home.tsx:
import Categories from "../components/Categories"
import Sorting from "../components/Sorting"
import PizzaBlock from "../components/PizzaBlock"
import Skeleton from "../components/PizzaBlock/Skeleton"
import Pagination from "../components/Pagination"

до:
import {Categories, Sorting, PizzaBlock, Skeleton, Pagination} from "../components"

27.5.2 Это рабочий способ, но есть способ лучше, мы можем также записать тот же reexport следующим способом, где всё работать будет также, но мы существенно сокращаем кол-во строк:

// [/src/components/index.ts]:

export {default as CartItem} from "./CartItem";
export {default as Categories} from "./Categories";
export {default as Header} from "./Header";
export {default as Sorting} from "./Sorting";
export {default as PizzaBlock} from "./PizzaBlock";
export {default as Skeleton} from "./PizzaBlock/Skeleton.tsx";
export {default as Pagination} from "./Pagination";

27.5.3 Также есть ещё вариант с подобной записью, но для него надо экспортировать функции компонентов напрямую, т.е. export const ComponentName = () => { ... }, без "export default ComponentName" в конце:

export * from "./CartItem";
export * from "./Categories";
export * from "./Header";
export * from "./Sorting";
export * from "./PizzaBlock";
export * from "./PizzaBlock/Skeleton.tsx";
export * from "./Pagination";
